h2. Searchable Product Depository (SPUD)

The _Searchable Product Depository_ (SPUD) is the EarthScope DMC's primary data product management system. Complementing the DMC's SEED and assembled data archives, which contain time series recordings, the SPUD system primarily contains derivative data products of other types (images, movies, etc.) created either at the DMC or by members of the community. "Access SPUD »":/spud/

h2. PyWEED

_PyWEED_ is a map-based seismogram request tool. It enables the user to select events and stations and then download the data to their computer. PyWEED is platform-independent and allows the user to access information from remote data centers, using the "EarthScope Web Services":http://service.iris.edu/. "Read More »":/ds/nodes/dmc/manuals/pyweed/
